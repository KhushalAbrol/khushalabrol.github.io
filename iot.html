Specilization:An Introduction to Programming the Internet of Things (IOT)

This specilization consists of 5 courses and a capstone project which is peer graded.This course is available on Coursera. And this course is by UCI.These are the courses in this course:

  1. Introduction to the Internet of Things and Embedded System(Course 1)
  2. The Arduino Platform and C Programming(Course 2)
  3. Interfacing with the Arduino(Course 3)
  4. The Raspberry Pi Platform and Python(Course 4)
  5. Interfacing with the Raspberry Pi(Course 5)
  6. Programming for the Internet of Things Project(Capstone)
